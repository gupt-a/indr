Requirement

For now I only wrote this with Linux servers in mind so if you want to run this on a Linux server just make sure you have Boost and OpenSSL installed, then build this with CMake. All other third-party code is included in this repository itself so you don't need to install anything else.

Supported platforms and compilers

While I have tried my best to use cross-platform code wherever possible, indr is still very, very alpha. I haven't even tried to compile it on a different system yet. So yeah, getting it work on other platforms is far in the future. It seems to work pretty well on the stable Debian release with GCC 7.5.0, though.
